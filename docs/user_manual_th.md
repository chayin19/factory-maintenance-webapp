# คู่มือใช้งานระบบแจ้งซ่อมโรงงาน

## หน้าแรก

1. เปิดลิงก์ Web App
2. เลือกเครื่องจักรจากปุ่มขนาดใหญ่
3. เครื่องจักรถูกจัดกลุ่มตามโรงงานและประเภทเครื่อง

ปุ่มเมนูด้านบน:

- `หน้าแรก` กลับหน้าเลือกเครื่องจักร
- `หน้าช่าง` เปิดงานซ่อมสำหรับช่าง
- `Dashboard` เปิดแดชบอร์ดหัวหน้างาน

## การแจ้งซ่อม

1. เลือกเครื่องจักร
2. เลือกอาการเสีย
3. ถ้าไม่พบอาการ ให้เลือก `อื่น ๆ / ไม่อยู่ในรายการ`
4. ถ้าไม่แน่ใจ ให้เลือก `ไม่แน่ใจ ให้ช่างประเมิน`
5. เลือกระดับความรุนแรง:
   - `หยุดเครื่อง`
   - `ผลิตได้แต่เสี่ยงเสีย`
   - `แจ้งตรวจสอบ`
6. กรอกรายละเอียดเพิ่มเติมถ้ามี
7. ใส่ลิงก์รูปภาพถ้ามี
8. กด `ส่งใบแจ้งซ่อม`
9. ระบบจะแสดงเลขที่งาน เช่น `REQ-20260424-0001`

## หน้าช่าง

1. เปิดเมนู `หน้าช่าง`
2. กรอกชื่อช่าง
3. เลือกงานที่ต้องทำ
4. กด `รับงาน`
5. เมื่อเริ่มทำงานจริง กด `เริ่มซ่อม`
6. เมื่องานเสร็จ กด `ปิดงาน`
7. กรอกข้อมูลปิดงาน:
   - ประเภทงาน
   - สาเหตุหลัก
   - การแก้ไข
   - เครื่องหยุดหรือไม่
   - บันทึกการซ่อม
8. กด `ปิดงาน`

ระบบไม่อนุญาตให้ปิดงานก่อนรับงานและเริ่มซ่อม

## แดชบอร์ดหัวหน้างาน

หน้าแดชบอร์ดแสดงข้อมูล:

- งานเปิด
- งานกำลังซ่อม
- งานที่ปิดวันนี้
- เวลาหยุดเครื่องวันนี้
- เครื่องจักรที่แจ้งซ่อมสูงสุด 5 อันดับ
- อาการเสียที่พบสูงสุด 5 อันดับ
- งานที่รอเกิน 30 นาที

## คำแนะนำ

- ผู้ปฏิบัติงานควรเลือกอาการที่ใกล้เคียงที่สุด
- ถ้าไม่แน่ใจ ให้เลือก `ไม่แน่ใจ ให้ช่างประเมิน`
- ช่างควรกด `รับงาน` และ `เริ่มซ่อม` ตามเวลาจริง เพื่อให้เวลารอและเวลาซ่อมถูกต้อง
